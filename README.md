## 🚀 Quick Start

```bash
# Clone this repository
git clone https://github.com/libxai/libgantt-ai-landing.git

# Navigate to the project
cd libgantt-ai-landing

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
